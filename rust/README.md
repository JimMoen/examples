# Rust Examples

## Install Dependencies

Install Rust:

``` shell
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh -s -- --no-modify-path --default-toolchain none -y
```

Introduce Cargo into the PATH environment variable:

``` shell
. "$HOME/.cargo/env"
```

## Run

``` shell
cargo run
```
